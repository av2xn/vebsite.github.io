Features

Arch-Like Philosophy

Inspired by the simplicity and DIY approach of Arch Linux, tailored for Android devices.

ADB-First

ADB access enabled by default — power users take control from the start.

No Bloatware

Ships clean. No preinstalled apps, just the core essentials.

Super Fast

Designed to be lightning fast, minimal and resource-friendly.

Open Source

Completely open and community-driven. Fork it, build it, tweak it.

Customizable

Your ROM, your rules. Configure everything via ADB and shell tools.

Disclaimer: TixOS is not intended for daily use. This project is specifically designed for IoT and server-class devices, and it includes no graphical user interface (GUI) applications or proprietary drivers. It is a command-line interface (CLI) based ROM, intended for developers and advanced users who require full control over their system.
